### 🛡️ **The Privacy Wall**
Unlike traditional health apps, **Sakhi is offline-by-design.** 
* **Zero Cloud:** No data ever leaves your device. No database. No tracking.
* **On-Device ML:** Physiological trends are computed locally using private autocorrelation engines.
* **Clinical-Grade:** Transforms daily agony into structured reports ready for your specialist.

### ⚙️ **Technical Core**
Built for high-integrity performance and data sovereignty.
* **Core:** Rust-powered backend for memory safety and speed.
* **Inference:** Lightweight, local ML models for symptom pattern recognition.
* **Hardened:** Obfuscated binary to prevent unauthorized telemetry or tampering.

### 📥 **Getting Started**
1. Download the latest `.apk` from the [Releases](https://github.com/yourlink/releases) page.
2. Enable "Install from Unknown Sources" on your Android device.
3. Start logging. Your data stays yours forever.
